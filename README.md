# MichaelGPT

Современный AI-чатбот с динамичным дизайном и плавными анимациями.

## 🚀 Функции

- **Система аккаунтов**: Регистрация и вход
- **Чаты**: Создание, переименование, удаление чатов
- **Сообщения**: Отправка, копирование, история переписок
- **AI-интеграция**: Подключение к GROQ API (Llama 3.3)
- **База данных**: SQLite для хранения пользователей, чатов и сообщений
- **Современный UI**: Адаптивный дизайн, анимации, тёмная тема

## 📁 Структура проекта

```
Site/
├── server/          # Бэкенд (Node.js + Express)
│   ├── server.js    # Основной сервер
│   ├── database.js  # База данных SQLite
│   └── middleware/  # Middleware (авторизация)
└── client/          # Фронтенд (React + Vite)
    └── src/
        ├── pages/   # Страницы приложения
        ├── contexts/# React контексты
        └── utils/   # Утилиты (API клиент)
```

## 🛠 Установка и запуск

### 1. Запуск бэкенда

```bash
cd server
npm install
npm start
```

Сервер запустится на порту **5000**.

### 2. Запуск фронтенда (в новом терминале)

```bash
cd client
npm install
npm run dev
```

Фронтенд запустится на порту **3000**.

### 3. Открыть в браузере

Перейдите по адресу: **http://localhost:3000**

## 🔑 API Ключи
